ALGORITMOS

Un algoritmo se puede definir como una secuencia de instrucciones que representan un modelo de solución para determinado tipo de problemas. O bien como un conjunto de instrucciones que realizadas en orden conducen a obtener la solución de un problema.

Las características principales que debe tener un algoritmo son:

  1. Finito: Debe culminar la solución.
  2. Ordenado: Debe tener una secuencia para seguir los pasos secuencialmente.
  3. Preciso: Debe ser claro y no tener diferentes interpretaciones, sino solo una.

• Definición
Un problema es una situación donde se requiere encontrar uno o varios elementos desconocidos, los elementos desconocidos pueden ser figuras, diagramas, números, decisiones, posiciones, algoritmos u otras cosas que cumplen relaciones y/o condiciones con uno o varios elementos desconocidos.


• Etapas Para La Solución De Problemas
Para resolver un problema es necesario seguir las siguientes etapas:

  1. Datos de Entrada: Son los datos que provee el problema o que son necesarios para realizar los cálculos.
  2. Datos de Salida: Son los datos que se desean calcular en el problema.
  3. Proceso: Son las operaciones, verificaciones que deben realizarse con los datos de entrada para calcular los datos de salida.
